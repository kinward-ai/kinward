GETTING STARTED

From download to first conversation

Total time: 5–10 minutes (including AI model download)

1
Install Prerequisites
~2 minutes
🦙
Install Ollama You

Download Ollama from ollama.com. On Mac, open the .dmg and drag it to Applications. On Windows, run the installer. Once installed, Ollama runs quietly in your menu bar — no configuration needed.

⯈
Install Node.js You

If you don't already have it, download Node.js LTS from nodejs.org. Run the installer with default settings.

Already have it? Check with:

> node --version
v22.15.0
2
Download & Launch Kinward
~1 minute
📥
Get the code and launch You

Clone or download from github.com/kinward-ai/kinward, install dependencies, and launch the Electron app:

> git clone https://github.com/kinward-ai/kinward
> cd kinward
> npm install
> npm run electron:dev

Kinward is starting up!
Look for the 🛡️ shield icon in your system tray
Tip: First launch takes 30–60 seconds to compile native modules. After that, startup is fast.
3
Setup Wizard
~2–3 minutes
🔮
The wizard walks you through 4 quick steps You

The Kinward app window opens automatically. The setup wizard guides you through everything:

STEP 1
Name Your AI
Give your family AI a name and tagline
STEP 2
Family Profiles
Pick a preset (Just Me, Couple, Family of 4) and customize
STEP 3
Install a Model
Kinward recommends the best model for your hardware
STEP 4
You're Ready
Start chatting with your family's AI
About model downloads: In Step 3, Kinward recommends an AI model based on your hardware. Downloading a model is a one-time thing — Gemma 4 9B is about 5.5 GB, Mistral Small 24B is about 14 GB, and models range from 2.5 GB to 17 GB depending on what you choose. This is usually the longest part of setup. After that, everything runs locally with no internet needed.
4
Start Chatting
You're done!
💬
Your family's AI is live Ready

Pick a conversation mode and start talking. Everything stays on your machine — no cloud, no accounts, no data leaving your home.

📱
Connect from any device

Everyone in the house can connect from their phone, tablet, or laptop. Just open a browser and go to the address shown in the app (something like http://192.168.x.x:3210). On your phone, tap "Add to Home Screen" for an app-like experience.

📚
Things to try

Upload a document — drag a PDF, text file, or photo into the chat. Ask questions about it. Key facts are remembered automatically.

Teach it about your family — mention allergies, school schedules, pet names. It extracts and stores these facts for future conversations.

Manage your models — go to Settings → AI Models to install, switch, or remove models. Try Gemma 4 9B, Gemma 4 27B, Mistral Small 24B, Llama 3.1 8B, and more.

Try different modes — switch between General, Kids, Research, and Creative for different conversation styles.

↻
Stopping & Restarting
Good to know
⏹️
How to stop Kinward

Click the 🛡️ shield icon in your system tray and select Quit. Or simply close the Kinward window.

▶️
How to restart Kinward

Run npm run electron:dev again from the Kinward folder. All your profiles, conversations, and memories are saved — nothing is lost.

Tip: Kinward stores all data in your Application Support folder — nothing is lost between sessions. Ollama is auto-detected and managed for you.
?
Troubleshooting
Common questions

"The app window is blank" — Wait 10–15 seconds for the server to start. You can also check the 🛡️ shield icon in your system tray to confirm Kinward is running.

"Lumina is sleeping" — Make sure Ollama is running. Look for the llama icon in your menu bar. If it's not there, open Ollama from your Applications folder.

"npm install failed" — Make sure you have Node.js 20 or 22 LTS installed. Kinward does not support Node.js v23 or newer.

"Port 3210 is already in use" — Kinward tries to free the port automatically. If it can't, run: lsof -ti:3210 | xargs kill -9

Found a bug? — Email hello@kinward.ai or open an issue on GitHub.
